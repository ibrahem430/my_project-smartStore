# 🛒 TechSmart

TechSmart is a full-stack e-commerce web application for buying and selling electronic devices and accessories. The project includes a customer-facing website, an admin dashboard, and a backend REST API for managing products, categories, users, and advertisements.

---

## 🚀 Features

### Customer

- User Registration
- User Login
- Browse Products
- Browse Products by Categories
- View Product Details
- Shopping Cart

### Admin

- Admin Login
- Product Management
  - Add Products
  - Update Products
  - Delete Products
- Advertisement Management

---

## 🎯 Project Objectives

- Develop a modern full-stack e-commerce platform.
- Build a RESTful backend using Node.js and Express.
- Separate customer and admin interfaces.
- Manage data using PostgreSQL.
- Provide a simple and user-friendly shopping experience.

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Customer Frontend

```bash
cd project_smartStore
npm install
npm run dev
```

### Admin Dashboard

```bash
cd my-react-app
npm install
npm run dev
```
